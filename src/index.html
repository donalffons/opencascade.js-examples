Examples

  • Polygon
    • Creates a simple polygon and renders it.
  • Bottle & File Loader Example - Version 01
    • Contains the OpenCascade bottle example with adjustable parameters.
    • Contains an example on how to load STEP and IGES files by using Emscripten's file system emulation.
  • Bottle & File Loader Example - Version 02
    • Similar to the previous example, but with a slightly different tessellation algorithm. This produces one ThreeJS object per face (vs a single one in version 01) and uses ThreeJS.BufferGeometries (vs plain ThreeJS.Geometries).
